## 참고
- 이름이 다르면 그대로 저장하고 **알려만 줘** — 내가 덱 코드에서 경로 맞춰줄게.
- 이미지 없으면 점선 플레이스홀더 박스가 자동으로 표시됨(나중에 끼우면 됨).
- P11(개발 과정)은 4컷이 기본인데, 1장 합본 이미지로 줘도 돼 — 그러면 레이아웃만 바꿔줄게.
