# PRD — 대진대학교 AI캠퍼스 (작성 예시)

> 강사가 실제로 이 앱을 만들 때 쓴 기획 문서입니다. **이렇게 채우면 됩니다.**

## 1. 개요
| 항목 | 내용 |
|---|---|
| 제품명 | 대진대학교 AI캠퍼스 |
| 한 줄 정의 | 바이브코딩 캠프 참가자를 위한 **강의 자료 + 결과물 공유 게시판** PWA |
| 플랫폼 | 모바일 우선 웹앱 (PWA, 홈 화면 설치형) |
| 타깃 | 캠프 참가자(비개발자) 약 30명 |

## 2. 목적 (왜 만드나)
1. **강의 동반** — QR로 입장해 오늘의 흐름·치트시트·도구 링크를 한곳에서 본다.
2. **결과물 공유** — 각자 만든 앱을 올리고 서로 구경 → 마무리에 "다 같이 보기".

## 3. 핵심 기능
### F1. 강의 자료 (로그인 불필요)
- 오늘의 흐름 · 프롬프트 치트시트 · 도구 링크 모음

### F2. 인증 (학번 로그인)
- 학번 + 비밀번호로 가입/로그인 (이름·과 입력)

### F3. 결과물 게시판
- 글 작성: 앱 URL + 스크린샷 + 제목/한줄설명
- 좋아요(사용자당 1회) · 최신순/인기순 정렬
- 열람은 누구나, 작성은 로그인 사용자

## 4. 화면 흐름 (요약)
홈/자료 → 게시판 목록 → 글 상세 → 글쓰기 → 로그인/회원가입

## 5. 범위 (Scope)
- **In:** F1 ~ F3
- **Out(오늘 안 함):** 결제, 댓글, 알림, 팔로우, 검색 — 결제는 "이런 게 있다"만 언급

## 6. 데이터
- 사용자(학번·이름·과) / 게시글(URL·이미지·제목·설명·작성자) / 좋아요
