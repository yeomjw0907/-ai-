```

먼저 docs/지침.md와 docs 폴더 안의 문서들을 모두 읽고, 내가 만들 앱을 기획할 수 있도록 인터뷰를 진행해줘.

질문은 한 번에 3~5개씩만 해줘.
내가 답하면 그 내용을 바탕으로 아래 문서를 채워줘.

- docs/서비스-기획안.md
- docs/디자인-시스템.md
- docs/PRD.md
- docs/유저플로우.md
- docs/기능정의서.md

아직 코드는 만들지 말고, 문서 초안과 다음에 구현할 1순위 화면만 제안해줘.
```
