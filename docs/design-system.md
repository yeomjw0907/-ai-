# 디자인 시스템 — 대진대학교 AI캠퍼스

> 모바일 우선. 깔끔하고 신뢰감 있는 캠퍼스 톤. Cursor에 줄 스타일 기준.

## 1. 톤 & 무드
- 키워드: **깔끔 · 또렷 · 캠퍼스 · 신뢰**
- 카드 중심 갤러리, 여백 넉넉, 큰 터치 영역(모바일)

## 0. 브랜드 / 로고
- 공식 대진대학교 엠블럼 사용: `../logo_color.png` (블루 엠블럼 + 레드/그레이 워드마크)
- 헤더·스플래시·PWA 아이콘 기준 자산. 색 토큰(Primary 블루 / Accent 레드)은 로고에서 파생.
- 추후 PWA용 192/512 아이콘은 이 로고 기반으로 생성.

## 2. 색 (Color)
| 용도 | 토큰 | 값 |
|---|---|---|
| Primary (대진 블루) | `--primary` | `#0B4DA2` |
| Primary 강조 | `--primary-strong` | `#083B7D` |
| Accent (좋아요 ♡) | `--accent` | `#FF4D6D` |
| 배경 | `--bg` | `#F7F8FA` |
| 카드/표면 | `--surface` | `#FFFFFF` |
| 본문 텍스트 | `--text` | `#1A1D21` |
| 보조 텍스트 | `--text-muted` | `#6B7280` |
| 경계선 | `--border` | `#E5E7EB` |

## 3. 타이포 (Typography)
- 폰트: **Pretendard** (한글 가독성, 웹폰트)
- 스케일: 제목 20–24 / 소제목 16–18 / 본문 14–16 / 보조 12–13
- 굵기: 제목 700 / 본문 400–500

## 4. 컴포넌트 규칙
| 요소 | 규칙 |
|---|---|
| 버튼(기본) | primary 배경, 흰 글씨, radius 12, 높이 48(모바일 터치) |
| 버튼(서브) | 외곽선 + primary 글씨 |
| 카드 | surface 배경, radius 16, 옅은 그림자, 패딩 16 |
| 결과물 카드 | 상단 스샷(16:10) + 제목 + 한줄설명 + 작성자 + ♡카운트 |
| 인풋 | radius 12, 경계선 border, 포커스 시 primary |
| 하단 탭바 | 3탭(자료·게시판·내정보), 활성 primary |
| FAB(글쓰기) | primary 원형, 우하단 고정 |

## 5. 레이아웃
- 컨테이너 최대폭 `480px`(모바일 우선), 가운데 정렬
- 간격 단위 4px 그리드 (4·8·12·16·24)
- 이미지: object-cover, 비율 고정으로 깨짐 방지

## 6. 레퍼런스 (느낌 참고)
- Mobbin / 모던 커뮤니티·갤러리 앱의 카드 피드
- "예쁜 껍데기"보다 **또렷한 정보 위계**가 우선
