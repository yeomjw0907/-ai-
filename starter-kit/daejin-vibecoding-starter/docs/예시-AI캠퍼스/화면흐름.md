# 화면 흐름 — 대진대학교 AI캠퍼스 (작성 예시)

## 화면 목록
| 화면 | 하는 일 | 로그인 필요? |
|---|---|---|
| 홈/자료 | 오늘 흐름·치트시트·도구 링크 | 아니오 |
| 게시판 목록 | 결과물 카드 갤러리 + 최신/인기순 토글 | 아니오 |
| 글 상세 | 스크린샷·링크·좋아요 | 아니오 |
| 글쓰기 | URL·스샷·제목·한줄 입력 후 게시 | 예 |
| 로그인/회원가입 | 학번·비밀번호 (가입 시 이름·과) | - |

## 화면 이동 흐름
```
홈/자료
 └─[게시판 탭]→ 게시판 목록
                 └─[카드 클릭]→ 글 상세 ─[♡ 좋아요]
 로그인 ──→ 글쓰기 ──→ 게시 ──→ 게시판 목록으로
```

## 각 화면 구성 요소
- **홈/자료:** 상단 환영 카드, 아코디언(레퍼런스·세팅·용어), 하단 탭
- **게시판 목록:** 정렬 토글, 결과물 카드(썸네일·제목·좋아요수)
- **글 상세:** 큰 스크린샷, 앱 링크 버튼, 좋아요, 작성자
- **글쓰기:** URL 입력, 이미지 업로드, 제목/한줄설명, 게시 버튼
